Install by extracting to anywhere.

Making Mods:
1. Click on "File" then "Open Archive"
2. Browse to a texture or file
3. Right click on the file then "Export" to
   export the file to the harddrive
4. Right click on the file then "Import" to
   import a new version of the file
4b. File size MUST be the same!
5. Play with your new mod.

Mod Manager
1. Place mods inside the mod folder
   Follow the existing mod as an example
2. Include a preview.png and desc.txt
3. Before you can do anything you MUST set
   the path to the Data folder!
4. The "Backup" button backs up all of the
   default .dat archives to a backup folder
5. The "Revert" button restores the backups
   to the client.
6. The "Apply" button applies checked mods.
